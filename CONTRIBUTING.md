# Contributing to webext-badge

Thank you for your interest in contributing to webext-badge!

## Getting Started

1. **Fork the repository** — Click the "Fork" button on GitHub
2. **Clone your fork** — `git clone https://github.com/YOUR_USERNAME/webext-badge.git`
3. **Install dependencies** — `pnpm install` (or `npm install`)
4. **Create a branch** — `git checkout -b my-feature`

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build the project
pnpm build
```

## Making Changes

1. Make your changes in your feature branch
2. Ensure tests pass: `pnpm test`
3. Commit your changes with a descriptive message
4. Push to your fork
5. Open a Pull Request against the `main` branch

## Pull Request Guidelines

- Describe what your PR changes and why
- Ensure all tests pass
- Update documentation if needed
- Be responsive to feedback
